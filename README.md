## 📌 Sobre o Projeto

O **ZAV Finances** é uma API de controle financeiro pessoal desenvolvida com foco em **arquitetura limpa**, **boas práticas de código** e **testes automatizados**. O projeto nasceu como uma versão robusta e escalável de um sistema financeiro real, com separação clara entre regras de negócio e infraestrutura.

---

## ✅ Funcionalidades

- 🔐 Autenticação com JWT e refresh token
- 💸 Controle de receitas e despesas
- 🏷️ Categorias personalizadas por usuário
- 🎯 Metas financeiras com acompanhamento de progresso
- 📊 Dashboard com resumo mensal e gastos por categoria
- 🔍 Listagem com filtros avançados e paginação
- 🗃️ Histórico de transações preservado mesmo ao deletar categorias
